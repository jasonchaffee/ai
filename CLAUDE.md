# CLAUDE.md - ai

Unified configuration for AI coding assistants (Claude, Gemini, ChatGPT, Copilot).

## Claude Assets (`claude/`)
- `commands/` - Slash commands (Markdown + YAML frontmatter)
- `subagents/` - Specialized agent definitions (Markdown + YAML frontmatter)
- `skills/` - Packaged skills (`SKILL.md` + optional `references/`, `scripts/`)
- `settings/statuslines/` - Custom statusline implementations
